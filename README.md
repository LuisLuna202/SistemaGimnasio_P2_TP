# SistemaGimnasio_P2_TP
Sistema que permita registrar a sus usuarios, crear rutinas de entrenamiento personalizadas (por ejemplo: bajar de peso, ganar masa muscular, mejorar condición física). y asignar entrenadores a cada usuario, buscar usuarios, rutinas y entrenadores
